# 城市治理模拟：iPad 安装说明

本目录是一套完整的 PWA 静态应用。它必须通过 HTTPS 网站打开，不能直接把压缩包中的 `index.html` 添加到主屏幕。

## 部署

将本目录中的全部文件原样上传到支持 HTTPS 的静态网站，例如 GitHub Pages。必须保留以下相对路径：

- `index.html`
- `manifest.webmanifest`
- `service-worker.js`
- `icons/`

## 安装到 iPad

1. 使用 Safari 打开部署后的 HTTPS 地址。
2. 点击 Safari 的“共享”按钮。
3. 选择“添加到主屏幕”。
4. 保持“作为 Web App 打开”开启，然后点击“添加”。
5. 从主屏幕启动“城市治理”，建议横屏游玩。

首次联网打开后，应用会缓存必要文件。之后可以离线启动，游戏存档保存在该 iPad 的本地应用数据中。

删除主屏幕应用或清理 Safari 网站数据可能同时删除本地存档。
